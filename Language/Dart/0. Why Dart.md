# 0. Why Dart

## Dart Native

- machine code JIT(Just In Time) and AOT(Ahead Of Time)
    - JIT
        - 작성한 코드의 결과를 바로 화면에 보여줌
    - AOT
        - 컴파일을 먼저하고 그 결과인 바이너리를 배포
- 개발중에는 JIT로 즉시 결과를 보여주고 배포 배포 후에는 AOT로 실제 기계어로 변환

## Null Safety

- 프로그램을 안전하게 만들어줌
- 개발자가 null을 참조할때 발생하는 문제를 해결
- 등 등…

- 따라서 속도의 향상을 야기할 수 있음
